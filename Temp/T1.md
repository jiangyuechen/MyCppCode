## 扩展欧几里得算法
已知 $a,b,c \in \mathbf{N}$ ，求解 $ax+by=c$ 的一组整数解 $(x,y)$。
```cpp
void exgcd(int a, int b, int& x, int& y)
{
    if (b == 0)
    {
        x = 1;
        y = 0;
        return;
    }
    exgcd(b, a % b, y, x);
    y -= a / b * x;
}
```
- 已知 $a,b \in \mathbf{N}$ ，求 $ax \equiv 1 \ (\mathrm{mod} \ b)$的最小正整数解 $x$。
```cpp
exgcd(a, b, x, y);
cout << (x + b) % b;
```
